JSalpha website

A back-end rendering engine. With the syntax of Template Literals ECMA-script

Download JSalpha:

npm install jsalpha

Official plugins

  • JSalpha-markdown It allows to insert the data of a markdown in a jsa template. For more information go to the docs

Functionality planning

  • JSalpha-yalm will be a plugin to accept YALM in a Jsa template

  • JSalpha-build it is already in progress. It would be a website builder based on jsa.

Get involved

There are several points on which you can participate.

  • Reporting an Issue You have a problem? A bug? Open an issue on GitHub! We will fix it.

  • Provide functionality to jsalpha or a plugin If you have an idea open an issue on GitHub. And tell us what you have in mind.
