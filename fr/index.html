Site web du JSalpha
Langue
English Français

Un moteur de rendu back-end. Avec la syntaxe des Template Literals ECMA-script

Télécharger JSalpha:

npm install jsalpha

Plugins officiels

  • JSalpha-markdown It allows to insert the data of a markdown in a jsa template. For more information go to the docs

Fonctionnalités prévues

  • JSalpha-yalm sera un plugin pour accepter YALM dans un modèle Jsa

  • JSalpha-build il est déjà en cours. Il s'agirait d'un constructeur de site web basé sur jsa.

S'impliquer

Il y a plusieurs points sur lesquels vous pouvez participer.

  • Signaler un bogue Vous avez un problème ? Un bug ? Ouvrez une issue sur GitHub! Nous allons le réparer.

  • Fournir une fonctionnalité à jsalpha ou à un plugin Si vous avez une idée, ouvrez une issue sur GitHub. Et dites-nous ce que vous avez en tête.
